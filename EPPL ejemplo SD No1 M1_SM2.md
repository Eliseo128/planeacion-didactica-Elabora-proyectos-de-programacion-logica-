  * Algoritmos
  * Lógica computacional
  * Introducción a IA (reglas, decisiones simples) (2 h)
* Ejemplo demostrativo en Python (1 h):

```python
# Simulación básica de toma de decisiones (IA simple)
temperatura = int(input("Ingresa temperatura: "))

if temperatura > 30:
    print("Recomendación: usar ropa ligera")
else:
    print("Recomendación: usar chamarra")
```
